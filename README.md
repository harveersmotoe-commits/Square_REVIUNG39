# Square_REVIUNG39
A split rectangular keyboard, cutting costs by using (Reviung39/41/30 keyboard PCB and FR4 plate), a pre-made PCB and plate. Using CAD I will create a case type object that fits onto the split board.

Base code has been added but will be tweaked after final product is created
